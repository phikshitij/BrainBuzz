This folder is reserved for UI screenshots for your BrainBuzz project README.

Recommended screenshots to include:
- teacher_dashboard.png: Teacher dashboard UI
- student_dashboard.png: Student dashboard UI
- chatbot.png: AI chatbot interface
- resources.png: Resource sharing/download UI
- grading.png: Grading interface (teacher view)

Place your PNG or JPG images here with the above filenames. The README.md will automatically display them if present.

Example structure:
Piazza/
├── backend/
├── screenshots/
│   ├── teacher_dashboard.png
│   ├── student_dashboard.png
│   ├── chatbot.png
│   ├── resources.png
│   └── grading.png
├── README.md
